# SovereignBrain

**VaultMesh Productivity Dashboard** - Alchemical Transformation Engine for Digital Sovereignty

## Overview

SovereignBrain is a web-based productivity dashboard aligned with VaultMesh philosophy, featuring alchemical phases, cryptographic concepts, and a unique operational framework.

## Features

### Core Modules

- **Dashboard** - Overview with stats, phase distribution, Six Organs status
- **Task Matrix** - Kanban-style board organized by alchemical phases (Nigredo -> Albedo -> Citrinitas -> Rubedo)
- **Knowledge Base** - Markdown notes organized by VaultMesh organs
- **Alchemical Journal** - Daily reflections with mood tracking and phase awareness
- **Calendar** - Monthly view with event/task integration
- **Bookmarks** - Research and reference link storage

### VaultMesh Integration

- Tasks/notes organized by the **Six Organs**:
  - Governance
  - Automation
  - Treasury
  - Federation
  - Psi-Field
  - Infrastructure
- **Alchemical transformation phases** as workflow states
- Cryptographic-style IDs (0x prefixed hex)
- VaultMesh timestamps
- Priority system aligned with operational criticality

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build
```

## Data Persistence

All data is stored in localStorage under the key `sovereignBrainState`. This includes:
- Tasks
- Notes
- Journal entries
- Bookmarks
- Calendar events

## Tech Stack

- React 18
- CSS3 with custom properties
- Local storage for persistence
- No external dependencies beyond React
